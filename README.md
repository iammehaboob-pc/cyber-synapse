# 🎮 Cyber Synapse

Cyber Synapse is a cyberpunk-themed memory matching game built with HTML, CSS, and JavaScript. Challenge your memory by matching pairs of cards while enjoying a futuristic interface, smooth animations, and responsive gameplay.

## ✨ Features

- 🧠 Memory card matching gameplay
- 🌃 Cyberpunk-inspired design
- ⚡ Smooth animations and transitions
- 📱 Fully responsive layout
- 🔄 Restart game functionality
- 🎯 Move counter and score tracking

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Play

1. Open the game in your browser.
2. Click on a card to reveal its symbol.
3. Match all pairs of cards.
4. Complete the game using the fewest moves.
